Setting Up Your Raspberry Pi with NOOBS (Raspbian)

Setting up a new Pi can seem like a long and daunting process. In this tutorial, you'll learn how to safely and efficiently setup your Raspberry Pi.

By Rohan Dharwadker

What you'll need

Recommended

First of all, I really recommend getting a starter kit that contains all the materials below. (I chose this one, which has NOOBS preloaded)

  • Raspberry Pi (I will be using the model 4 B)
  • Power Ssupply for Pi (5v USB-C for Pi 4; comes in most starter kits)
  • Display or monitor with HDMI input
  • Appropriate display cable (Micro-HDMI to HDMI for Pi 4 and most monitors)
  • USB keyboard and mouse (Preferable but not required to be connected to the same dongle)
  • Micro SD card with adapter (I recommend at least 32gb, but 16 will work.)

Optional

  • Case for Raspberry Pi
  • A starter kit that contains all the materials above (I chose this one, which is really great because it comes with a micro SD card that is preloaded with NOOBS)

Connecting the Display and the Peripherals

IMPORTANT! DO NOT POWER ON THE PI YET!

For this tutorial, I'll be using the Raspberry Pi Model 4 B, but you can use a different model with just a few changes. To start, you'll want to plug in your keyboard and mouse. I recommend plugging them into the USB 2.0 ports, so that the 3.0 ports remain availiable for things like usb flashdrives that can take advantage of the 3.0's faster speed. Next, plug in the the micro HDMI to the 'HDMI 0' port. If you are running dual monitors, you can plug in to both HDMI ports, but if your are only using one, make sure you plug your cable into the 'HDMI 0' port (The one closer to the power port).

Connecting the Ethernet Cable (Not Required)

Next, if you have one, plug in an ethernet cable to the ethernet port (located next to the USB 3.0 ports). This is not required if you have a Raspberry Pi 4 because it has on board Wi-Fi, but it will almost always be faster (although I can confirm that it does indeed work without the ethernet).

Connecting the Power Source

Before you plug in the power supply, make sure it is not plugged in on the other end, or if you have a switch on your power supply, make sure it is in the 'off' position. Now you can plug in the power supply to the USB-C power port.

Flashing NOOBS to the Micro SD

Skip this step if you have an SD card that is preloaded with NOOBS or Raspbian.

If your computer has a micro SD slot, insert the micro sd. If not, insert the micro sd into the adapter and insert the adapter into your computer. If you do not have a brand new micro SD, you'll need to format it so that it is completely cleared. You can do this with a software such as SD Card Formatter. Be sure you select the correct drive, as once formated, all data on that drive is unrecoverable. Once the SD card is cleared, you'll need to install NOOBS (New Out Of the Box Software). Head to https://www.raspberrypi.org/downloads/noobs/ and download it in any format. Once downloaded, extract or unzip the files and place them onto the micro SD. Keep in mind that this may take some time because you are loading an entire operating system onto a micro SD card.

Booting up the Pi

The next thing you'll need to do once you've installed NOOBS onto the micro SD is to install the Raspbian Operating System. Safely eject the micro SD card from your computer and insert it into the micro SD slot located on the bottom of the Pi. It only goes one way, so if it isn't going in, don't force it.

Now it's finally time to power on the Pi! With all the cables connected and the SD inserted, turn on the power by plugging in the power cable (and clicking the switch, if you have one). The first thing you should see on your display is a rainbow-gradient screen. On the Pi, you should see a solid red and flashing or solid green light.

Installing Raspbian & Setup

Once the Pi has booted, you will see the screen below.

If you do not have an ethernet plugged in you may not see all the download options, and you will want to click Wi-Fi networks and connect to one with a strong signal. Once you have established an internet connection, click 'Raspbian Full' so that an 'X' appears in the box next to it. Also make sure that the language at the bottom is correct. Once Raspbian Full is selected, click install. The installation process will take about 20 minutes, so be patient and ensure that your Pi does not lose power, as this may result in the corrupting or damaging of the SD card and/or the Pi.

Final Touches

Once the installation has finished, follow the OS directions to complete the final settings.

Congratulations, your Raspberry Pi now has Raspbian installed and is ready to use!
